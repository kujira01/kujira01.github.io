ハンズオンコマンド
1. Git Bashを開く。
2. Dockerコンテナを作成する：
docker container run -d -p 80:80 kujira1/mynginx:org
3. 上記で作成したコンテナを一覧表示し、IDを確認する：
docker container ls -a
4．実行中のコンテナ内でbash シェルを実行:
winpty docker container exec -it <コンテナ ID> bash
5. コンテナ内のhtmlを確認：
cat /usr/share/nginx/html/index.html
6．コンテナ内のhtmlを修正する：
sed -ie 's/nginx/<"name">/' /usr/share/nginx/html/index.html
7．コンテナのhtmlをブラウザから確認：
http://localhost:80/
8．Dockerコンテナから退出する：
exit
9．htmlを修正したコンテナのIDを確認：
docker container ls -a
10．IDを確認したコンテナを停止する：
docker stop <コンテナ ID>
11．コンテナからイメージをビルドする：
docker commit <コンテナ ID> kujira1/mynginx:<01_name>
12．コマンドラインからDocker Hubにログイン：
winpty docker login
Docker Id: kujira1
Password: kujirapw!
13．イメージをDocker Hubにプッシュ：
docker image push kujira1/mynginx:<01_name>
14．Azureにlogin：
az.cmd login
15．docker-machineを使用してAzureに VM を作成：
docker-machine create -d azure \
--azure-subscription-id $sub \
--azure-ssh-user azureuser \
--azure-open-port 80 \
--azure-size "Standard_DS2_v2" \
myvm
16．Docker コマンドの対象をAzure上に入れたDockerに変更する：
eval "$(docker-machine env myvm)"
17．DockerイメージからAzure上でコンテナを作成：
docker container run -d -p 80:80 kujira1/mynginx:<01_name>
18．Dockerコンテナを確認：
docker container ls -a
19．Azureのパブリック IP アドレスを取得：
docker-machine ip myvm
20．取得したIPアドレスをブラウザで確認。
21．Azure上の仮想マシンを停止：
docker-machine stop myvm
22．Azure上の仮想マシンを削除：
docker-machine rm myvm
